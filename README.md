# 🧠 Flutter Quiz App

A modern quiz application built with **Flutter** and **Dart** that allows users to answer multiple-choice questions, track their progress, and view their final score. The project follows a modular architecture with reusable widgets and demonstrates Flutter navigation, state management, and clean UI development.

---

## ✨ Features

* 📝 Multiple-Choice Questions
* 🎯 Score Calculation
* ➡️ Question-by-Question Navigation
* 🏆 Result Screen with Final Score
* 📱 Responsive & User-Friendly UI
* 🧩 Reusable Widgets
* ⚡ Smooth Navigation
* 🏗️ Clean Project Architecture

---

## 📂 Project Structure

```text
lib/
│
├── data/
├── models/
├── screens/
├── widgets/
└── main.dart
```

### Folder Description

* **data/** – Stores quiz questions and answer data.
* **models/** – Contains question model classes.
* **screens/** – Includes Home, Quiz, and Result screens.
* **widgets/** – Reusable UI components such as answer options and custom buttons.
* **main.dart** – Entry point of the application.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-quiz-app.git
```

### Navigate to the project

```bash
cd flutter-quiz-app
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
